# nRF52840 LED Blink Firmware

## Project Description
This embedded systems project is developed using SEGGER Embedded Studio targeting the Nordic Semiconductor nRF52840 SoC.

The firmware demonstrates basic GPIO configuration and LED control using Embedded C.

## Technical Details
- MCU: Nordic nRF52840
- Core: ARM Cortex-M4
- IDE: SEGGER Embedded Studio
- SDK: Nordic nRF5 SDK
- Language: C

## Features
- GPIO Initialization
- LED Toggle
- Delay using nrf_delay_ms()
- Bare-metal firmware structure
